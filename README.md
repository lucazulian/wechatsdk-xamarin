# wechatsdk-xamarin

Xamarin.Android and Xamarin.iOS WeChat SDK Binding
=======

This the Xamarin.Android and Xamarin.iOS Binding for Wechat SDK.

Please replace the APP_ID and Keystore respectively use your own one.
